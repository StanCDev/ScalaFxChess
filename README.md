# ScalaFxChess

## Description

Local two player game of chess.

![image info](./res/gameWindow.png)

## User Guide

Make sure SBT and scala 3 are installed.

1. `cd` into ScalaFxChess
2. type `sbt` in terminal
3. type `run` in sbt command line
